# MongoDB Training Management System

A MongoDB database project for managing students, mentors, tasks, attendance, and company placements.

## Collections

- **users** - Student information
- **mentors** - Mentor details
- **topics** - Training topics
- **tasks** - Student assignments
- **attendance** - Attendance records
- **company_drives** - Placement drives
- **codekata** - Coding practice records

## Setup

1. Start MongoDB:
   ```bash
   mongod
   ```

2. Connect to MongoDB:
   ```bash
   mongosh
   ```

3. Create database:
   ```javascript
   use training_institute
   ```

4. Insert sample data from `Collections and values.txt`

5. Run queries from `Queries .txt`

## Files

- `Collections and values.txt` - Database schema and sample data
- `Queries .txt` - MongoDB queries with outputs
- `Readme.md` - Documentation
